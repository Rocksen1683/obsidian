## Problem 
https://leetcode.com/problems/maximum-subarray/description/
## Solution 
``` python 
def maxSubArray(self, nums: List[int]) -> int:
	currentMax = 0
	globMax = float("-inf")
	
	for i in nums:
		#max for current subarray
		currentMax = max(currentMax + i, i)
		#max for all subarrays
		globMax = max(currentMax, globMax)
		
	return globMax
```
