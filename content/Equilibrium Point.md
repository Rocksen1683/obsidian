$(x^{-}, u^{-}) \in R^{n} \times R^{m}$ is an equilibrium point if 
$$
f(x^{-}, u^{-}) = 0
$$

This means that 
$$x' - f(x^{-}, u^{-}) = 0$$
$$x(t) = x^{-}$$
$$u(t) = u^{-}, t\in [t_{0}, t_{1}]$$
This means that $x(t) = x^{-}$ is a solution 

$$x' = f(x, u)$$
$$x(t_{0}) = x^{-} + \delta x^{-}$$
$$u(t) = u^{-} + \delta u(t)$$
